ASUS及び第三者たる当社のパートナーは、当社の製品やサービスをお客様に提供するために、クッキー（クッキーとはお客様の製品に設置された小さなテキストファイルであって、お客様のASUS製品やサービスのユーザー体験をお客様に合ったものにするためのものです）及びウェブビーコンといったクッキーと類似する技術を利用します。お客様がASUSのウェブサイトドメイン下にある当社のウェブサイト（マイクロサイトや特定の国／地域のバージョンを含みます）のいずれかを訪問するとき、そのようなASUSウェブサイトでは以下のクッキーや類似技術の一部又は全部を使用していることがあります。

クッキーを通じて収集された情報は、そのほとんどがお客様の製品にのみ保存され、ASUSに転送されることはありません。ごく一部の限られた状況においてのみ、お客様の情報であってクッキーを通じて収集されたものがASUSに共有されることがあります。例えば、お客様が当社の製品をASUSストアで購入したとき、当社はクッキーを使用しお客様のIPアドレスを、お客様がログインしたとき及びASUSストアで注文をしたときの両方の時点において収集し、それによって注文したユーザーがASUSストアにログインしたユーザーと同じであることを確認して、オンラインでの購入の安全性を確保します。

当社がクッキーをどのように利用するかについて
  1. お客様のオンライン体験をより豊かで完全なものにするため、当社はASUS製品やサービスに必須な以下のクッキーを利用します。
    • 機能 例
      サインアップと認証 当社では、お客様固有のサインアップID番号と認証情報をお客様の製品に保存するためにクッキーを利用します。クッキーにより、お客様は、次回以降ASUSの製品やサービスを利用する際、再度ログインすることなく閲覧し、サイト内のページ間を移動することができます。例えばASUSが提供するaticketクッキーがこれにあたります。
      お客様の好みや設定を保存する 当社は、クッキーを利用して、お客様の好みの言語、場所やフォントといった設定や好みをお客様の製品上に維持します。設定をクッキーに保存することによって、お客様が当社の製品やサービスを利用するたびにお客様の好みや設定を再適用する必要がありません。ASUSが提供する、current_siteクッキーやEntryPageクッキーがこれにあたります。
      ユーザーインプット機能 当社は、クッキーを利用してお客様がASUS製品やサービスに入力するデータを一時的に保存します。例えば、ASUSが提供するcountクッキーがこれにあたります。具体的には、お客様がASUSストアでショッピング体験を楽しむときに、そのようなクッキーが、クリックした製品や数量、入力した情報をお客様が記憶するのを支援します。
      セキュリティ 当社は、クッキーを利用して、お客様のオンラインでの購入活動のセキュリティを保護します。例えば、ASUSが提供するip_addressクッキーがこれにあたります。上記目的のため、お客様がASUSストアを通じて当社の製品を購入するとき、当社は、ASUSストアで注文をしたユーザーがASUSストアにログインしているユーザーと同一であることを確認するのに役立てるため、ASUSにお客様のIPアドレスを保存することがあります。
      ロードバランシング機能 当社は、お客様に当社ウェブサイトにおける安定的な閲覧体験を提供すべく、ロードバランシング機能のためにクッキーを利用します。例えばASUSが提供するBIGipServerNewクッキーがこれにあたります。
  2. 解析目的及びお客様に合った広告サービスやその他の機能を提供するため、当社は以下のクッキーを利用し、お客様がASUS製品やサービスを使用するにあたっての体験を最適なものにします。
    • 機能 例
      解析 当社では、クッキーを使って、お客様がASUS製品やサービスを利用される回数や長さの他、お客様がよく利用する部分や機能をカウントします。この情報は、当社がASUS製品やサービスのパフォーマンスや運用を解析してパフォーマンスを向上させ、新たな特徴、機能やサービスを開発するのに役に立ちます。例えばMIGO Corp.が提供するMIGOクッキー、Google Inc.が提供するGoogleTagManagerクッキー、Google Analyticsクッキーがこれにあたります。上記目的のため、お客様が当社のウェブサイトを閲覧するとき、当社はIPアドレスやお客様のASUSメンバーID等のお客様の個人情報を上記MIGOクッキー経由でASUSに保存することがあります。
      ターゲッティングと広告 当社は、クッキーを使って、自社のウェブサイトにYouTube動画を挿入するのに役立てています。Google Inc.が提供するそのようなクッキーによって、お客様は、YouTube動画を当社ウェブサイト経由で視聴できます。

      そのようなクッキーは、ある広告をお客様が見る回数を制限する他、広告キャンペーンの効果の評価を支援するためにも利用されます。ASUSは広告キャンペーンサービス、例えばHubrusの提供するHubrusDSP、Google Inc.の提供するGoogle AdWordsクッキーやGoogleDoubleClickクッキーなどを使ってクッキーを設置します。クッキーが収集する情報は、ASUSと広告キャンペーンサービス提供者との間でのみ利用されます。
      当社ウェブサイトに埋め込まれたYouTube動画の視聴 当社は、クッキーを使って、自社のウェブサイトにYouTube動画を挿入するのに役立てています。Google Inc.が提供するそのようなクッキーによって、お客様は、YouTube動画を当社ウェブサイト経由で視聴できます。
クッキーの設定の管理方法
  1. お客様は、お客様のインストールしたブラウザにアクセスしてクッキー（例えば、第三者のクッキー）の全部又は一部を許可、ブロック又は削除することで、クッキー環境を設定することができることにご留意ください。
  2. 一部の国では、お客様が初めてASUSウェブサイトを閲覧する際、当社は、ASUSウェブサイトの上部に設置されたバナーにおいて、当社がどのようにクッキーを使用するかについての簡単なご紹介を掲載する場合があります。お客様は、そのバナー経由で、自由に上記の第三者のクッキーを許可又はブロックする選択ができます。
  3. お客様がクッキーをブロックする選択をした場合、ASUS製品やサービスの全ての機能を利用することはできなくなる可能性があります。
  4. クッキーの設定の機能はお客様がインストールするブラウザの種類やバージョンにより異なる可能性があります。当社は、以下のとおり一般的にかつ広く使われている種類のブラウザを列挙するようにしています。お客様は、ご自身のクッキー設定を当該ブラウザ経由で管理する方法を理解するため、以下のリンク先を参照することができます（リンク先のコンテンツは英語です。お客様にとってより読み易いよう、リンク先において言語のオプションを見つけてお客様の好みの言語を選択してください）。また、お客様が以下のブラウザのいずれもお使いでない場合や以下のリンク先のコンテンツが除去されたかアクセス不能な場合は、更なる情報のために該当のブラウザの個人情報保護関連の声明やサポートページをご覧ください。お客様は、様々なブラウザを通じてお客様のクッキー設定を管理する方法を説明するhttps://www.aboutcookies.org/（本リンク先のコンテンツは英語です）を参照することもできます。

    • Microsoft Internet Explorer ブラウザ経由でお客様のクッキー設定を管理する方法：
    • https://support.microsoft.com/en-us/help/278835/how-to-delete-cookie-files-in-internet-explorer

ウェブビーコン

ウェブビーコンは、キャンペーンの効果を評価するためにウェブサイトや電子メール上に実装される多くの場合透明な1x1（ピクセル）のGIF又はPNG画像 です。ASUS又は当社のサービス提供者は、お客様が特定のページを訪れたかや、ASUS製品やサービスにおけるリンクをクリックしたかを知るためにウェブビーコンを利用することがあります。当社では、お客様がどのコミュニケーションのコンテンツをクリックし、又は読んだかを知るために、電子メールによるダイレクトメールやASUSニュースレターといった当社のマーケティングコミュニケーションにウェブビーコンを実装することがあります。当社は、ウェブビーコンから得た情報を当社ウェブサイトやASUS製品やサービスの改善に利用します。

セキュリティ

本段落は、ASUSがどのようにお客様の個人情報を保護するのかについてお伝えし、お客様においてご自身の個人情報を保護する方法についていくつかのご提案をするものです。

当社では、お客様の個人情報を不正アクセスや改変、開示又は破壊から保護するための予防措置を講じています。当社は、当社の情報収集、保管及び処理慣行並びに技術的及び組織的セキュリティ対策の他、当社がお客様の個人情報を保管しているシステムへの不正アクセスを防ぐための物理的なセキュリティ対策について、内部点検を実施しています。

ASUS及びその関連団体の異なる拠点間での個人情報送信は、当社の保護された広域ネットワークを通じて行われます。お客様が当社にお客様の個人情報を提供される場合、お客様の個人情報はオンラインでもオフラインでも保護されます。

但し、ASUSは、インターネット上における完全なセキュリティは保証できません。お客様の個人情報を不正アクセスから守るために、お客様には以下のことを推奨します：

お客様のASUSメンバー・アカウントを適切に保護すること。例えば：
  1. ASUSメンバー・アカウントにサインアップする際は英数字のパスワードを使用すること。
  2. ASUSメンバー・アカウントにログインするときにはお客様ご自身のアカウント名とパスワードを使用すること。また、お客様は、ご自身のアカウント名やパスワードを秘密にしておくことやご自身のASUSメンバー・アカウントの下で生じるすべての活動について単独で完全な責任を負います。
  3. 定期的にASUSメンバー・アカウントのパスワードを変更すること。
  4. お客様がご自身のASUSメンバー・アカウントのアカウント名やパスワードが悪用されたことに気が付いたときは、直ちに当社に連絡をすること。ASUSは当該アカウント名やパスワード（又はその一部）でお客様のASUSメンバー・アカウントにログインする許可を停止又は終了させ、お客様のASUSメンバー・アカウントに関連するお客様の個人情報を除去することがあります。
お客様のソフトウェアに最新の、入手可能なセキュリティアップデートを適用してお客様の製品を最新の状態に保ち、ウイルス/スパイウェア・スキャナー等のツールを利用すること。
ASUSの製品やサービスに悪影響を及ぼすような技術的脆弱性にお気づきになった場合は、遠慮なくprivacy@asus.com経由で当社にご連絡ください。
