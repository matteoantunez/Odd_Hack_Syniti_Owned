•  华硕隐私权保护政策

 华硕电脑股份有限公司及其关系企业(下称「我们」或「华硕」)尊重并承诺保护您的个人隐私，我们致力遵循关于保护您个人隐私及个人信息安全之相关法令要求。华硕隐私权保护政策及我们针对部份华硕产品及服务所制定的个人信息相关声明(下称「本政策」)将帮助您了解我们如何搜集、使用及保护我们透过华硕产品及服务自您所取得的个人信息，我们亦将说明我们如何揭露、传输您的个人信息及其对象。若您为未成年人，您应于您的父母亲(或监护人)阅读并了解本政策后方得使用本服务。

若您使用本服务、您或您的父母亲(或监护人)同意将您的个人信息提供予我们，皆将被视为您或您的父母亲(或监护人)同意本政策。


1.我们所搜集的信息及搜集的方式

此段介绍华硕透过我们的产品及服务，搜集了您的哪些信息，以及我们将如何使用您的信息。

当您使用华硕产品及服务（如：华硕的电脑、软件、官方网站、客户支持服务等）时，我们可能会搜集您因为使用华硕产品及服务而提供给华硕的个人信息。
以下为整体而言，华硕透过各式各样的产品及服务，所可能搜集的个人信息项目以及搜集个人信息原因的汇整说明。请知悉我们将仅会透过您实际使用的华硕产品及服务，依该华硕产品及服务的特定信息搜集目的来搜集您的个人信息。另外，在某些国家，为了避免不当地搜集及使用未成年人的个人信息，我们可能需请您额外提供您的年龄或您父母亲（或监护人）的联络信息以取得其同意。此外，当您使用华硕产品及服务时，我们亦可能会搜集无法辨识您个人身分的以下匿名信息。
当您使用华硕产品及服务时，您不一定要依照我们的要求提供个人信息，但如果您选择不提供，会导致我们在许多情况下无法为您提供产品或服务，或是无法回应您提出的问题。

1.1 华硕所搜集的个人信息

个人信息为可以直接或间接地识别您个人身分的信息，例如：您的姓名、电子邮件地址、IP地址等。华硕会在取得您的同意后，才会搜集您的以下个人信息：

  1. 当您欲注册ASUS会员时，您需要提供您真实、正确、最新且完整的注册信息，包括您的电子邮件地址、所在的国家/地区、在部份国家会需要您提供年龄；如您选择透过您的社群账号（如：微博账号、QQ账号、微信账号）连结ASUS会员，该社群账号提供方(如：微博的公司、QQ的公司、微信的公司)在您同意的情况下，将可能把您于该社群账号所设定之账号信息(如：电子邮件、电话、姓名、昵称及生日)分享给华硕。另外，当您登入「华硕账号」（进入「华硕账号」的路径：http://www.asus.com/entry.htm→请点选网页右上角的「登入」→请输入您ASUS会员的账号/密码→ 进入「华硕账号」）时，您可自由选择在「个人设置」中提供其他个人信息（如大头照、性别、地址、职业）；亦可在体验ASUS会员相关服务（如：「产品注册」、论坛服务)）时，自由选择提供与该服务有关的个人信息（如：提供产品序号以完成产品注册、在我们的论坛发布或上传包含您个人信息的文章及图片）。
  2. 当您向我们购买华硕产品(如：在ASUS Store购买华硕产品)、部份付费服务时，您可能需要提供您的姓名、邮件地址(包含邮政编码)、电话号码、电子邮件地址、信用卡号码或其他付款信息。另外，当您向我们寻求客户支持服务(如：产品维修)时，除了前述信息，我们可能需请您额外提供您的产品的识别信息（如：产品序号、IMEI码）。
  3. 当您参加我们举办的营销活动或有奖竞赛时，依照不同类型的营销活动，您可能需要提供您的姓名、电话号码、电子邮件地址、出生日期、性别、产品的识别信息（如产品序号、IMEI码）、发票复印件(在某些国家的发票内容中，可能会包含您的姓名、地址等个人信息)等个人信息。若您是有奖竞赛的得奖者或是为了领取赠品，您可能需要额外提供您的邮件地址(包含邮政编码)及税赋申报所必要的信息（如户籍地址、身分证/护照的号码及复印件）等。
    此外，若我们将透过前述的营销活动提供您产品回馈金，除了您在前述参加营销活动或有奖竞赛时所提供的个人信息，我们可能需请您额外提供您的汇款信息。
  4. 当您使用我们有关健康与运动纪录的部份产品及服务时，您可能需要提供您的年龄、性别、身高、体重、其它身体质量信息如体温、脉搏、血压、心跳、腹部起伏及每日活动状况如行走步数、睡眠状况、卡路里消耗量、日志信息等。
  5. 当您使用华硕产品时，我们可能会搜集与您的产品相关连的产品识别信息，例如您的产品序号、因特网协议地址(IP Address)、MAC Address、IMEI码、Android ID码等。
  6. 当您使用华硕产品及服务时，我们可能会搜集与您的产品及服务相关连的位置信息，例如您的产品的GPS、WiFi的讯号、GPS定位位置、WiFi存取位置、基地台的序号、您的产品所在的国家、城市、时区、经纬度、高度、定位的精确度、您在行进时的产品移动速度、您使用产品及服务时所选择的国家与语言等。
  7. 当您透过各种方式联系华硕，例如拨打电话至华硕(如：华硕售后技术支持热线)、使用华硕在线客服平台、至华硕官方网站填写联络表单、寄送电子邮件至华硕时，我们将可能把我们与您的通话内容进行录音，或留存我们与您的对话纪录或往返的电子邮件。此外，若您亲临华硕的维修据点或办公大楼，我们将可能透过监视录影机对您的影像进行录影。前述的录音/录影/对话纪录，其内容可能包含您的个人信息。此外，当您使用我们有关机器人的部份服务或产品时，我们可能会搜集您使用应用程序与软件时所录制的语音识别录音文件与居家环境的录影文件。前述的录音/录影纪录，其内容可能包含您的个人信息。

1.2 我们如何使用您的个人信息

我们所搜集的个人信息将用于以下目的:

  1. 改善及提升华硕产品及服务。
  2. 使用者经验调查及分析，以开发、评估华硕的新产品、服务与功能。
  3. 完成ASUS会员账号创建、与体验ASUS会员相关服务（如：「产品注册」、论坛服务）。
  4. 对您向华硕所购买的华硕产品及服务，提供您寄送服务（包括购买证明、发票）、产品的软件升级、产品及服务的更新通知。
  5. 完成您对本服务相关营销信息之订阅，包括华硕相关新闻、优惠折扣、营销宣传活动等讯息。请注意，您可随时取消订阅。
  6. 寄送与您权益相关之重要通知给您，例如当适用于华硕产品及服务的条款、政策有所变更时。请注意，为保全您的权益，您将无法取消接收本项通知。
  7. 当您参加我们举办的营销活动或有奖竞赛时，用于确认您的参加资格、告知您活动动态、连系赠奖与寄送事宜、提供产品回馈金、税赋申报使用及于有活动接驳需求时，为您投保保险。
  8. 在您使用我们有关健康与健身纪录的部份产品及服务时，协助您记录、追踪、编辑、储存包括您的身体质量信息、每日活动状况及依照以上信息所计算出关于您的活动状况报告等的信息。当您选择与和您健康有关的第三方，例如您的家人、照护者、医疗服务人员等共享以上信息时，我们将协助您管理与以上信息相关的存取与修改权限。
  9. 当您联系华硕或亲临华硕的维修据点或办公大楼时，我们所搜集的您的个人信息及录音/录影/对话纪录信息是为了对您提供支持服务(如：维修服务、回复您的询问与查询)、提供客户关怀、满意度调查、获取意见回馈、维护您的权益和保障人员进出的安全。此外，当您使用我们有关机器人的部份服务或产品时，我们所搜集的您的录音/录影是为了让您的产品得依照需求抵达期望位置。
  10. 个人化营销，如透过第三方的广告性Cookies，提供符合您个人偏好的营销与广告内容、根据您过往使用华硕产品及服务的模式来推荐适合您的服务内容。
  11. 任何其他取得您同意的目的。

1.3 华硕所搜集的匿名信息及使用目的

匿名信息为无法直接或间接识别您个人身分的信息，例如：您的产品型号、软件的版本、发票日期等。当您使用华硕产品和服务时，我们可能会因为各种可能的目的来透过华硕产品和服务搜集以下的匿名信息。此外，当以下匿名信息与上述的个人信息产生连结时，我们亦会将以下匿名信息一并视为您的个人信息并进行适当地保护：

  1. 当您使用华硕产品及服务时，我们可能会搜集与您的产品及服务相关连的记录信息，例如您的产品型号、名称、品牌、制造商、料号、产品的硬件（如CPU、主机版）与操作系统的类型与版本、产品的出厂设定值、启用时间、固件的更新方式与结果与更新日期、记忆体与储存空间的大小、只读存储器（ROM）的相关信息(如ROM的类型、版本、水印、描述)、产品的相机解析度、产品机身颜色、使用之电信业者、连网状态、通话状态、待机状态、当机事件、惯用界面、浏览器的类型、版本与语言、诊断与流量信息、使用习惯、产品的GPS与WiFi版本、系统状态（如电池、CPU、RAM的使用状况）、网络速度、硬盘容量使用率、散热模组状态、当地时间等。
  2. 当您使用华硕产品及服务时，我们可能会搜集应用程序在您的产品及服务所产生的信息及您对应用程序的使用行为与互动方式，例如应用程序与软件的名称与版本、您安装与卸除的时间、您登入与登出的次数与时间、使用频率与次数、您开启与关闭的次数与时间、您偏好之应用程序与软件类型、使用习惯设定、应用程序与软件的更新版本与结果等。
  3. 当您向我们寻求客户支持服务(如：产品维修)、参加我们举办的营销活动时，我们可能需请您提供产品的购买信息（如发票购买日期、购买店家）。


2. 个人信息的保留期间

此段介绍华硕保留您的个人信息的期间。

我们将于完成本政策内所述使用目的之必要期间保留您的个人信息，除非有以下法定或其他必要的情况下，我们才会保留更长的期间：如，为了客户关系管理，在适当合理的期间内会保留您的个人信息、或为遵循税法或其他法律规定，在依法所要求的特定期间内保留您的个人信息、或为配合政府或司法机关调查或诉讼等要求。此外，若您一开始同意华硕向您搜集个人信息，但之后希望撤回此同意，在您撤回同意之后，我们便不会继续搜集您的个人信息，只会保留我们在您撤回同意之前所搜集的个人信息。


3. 我们揭露您个人信息的对象

此段介绍华硕在特定的情形及限缩的范围内，可能会将您的个人信息分享给第三方。

除非有以下任何一种情形，否则华硕不会将自您搜集得的个人信息揭露予其他第三方:

3.1 取得您的同意

  • 我们会在取得您的同意后，才可能向华硕以外之其他第三方揭露、分享您的个人信息。

3.2 合作伙伴

  • 我们可能将自您搜集得的个人信息经汇总并去识别化成为具不可逆性且无法辨识您的统计信息后始揭露予我们的合作伙伴，例如向您提供符合您个人喜好广告营销与宣传内容的合作伙伴们(如：广告性Cookies的第三方合作伙伴)。

3.3 服务提供商

  • 我们也会在本政策所提及之目的范围内，将您的个人信息的必要项目揭露、分享给我们的服务提供商，以利其提供服务给您，例如协助我们发送营销信息及举办营销活动的营销活动公司、运送您所购买或维修的产品的货运公司、处理付款事宜的金流处理商，以及提供您客服支持服务(像是华硕电话客服、华硕在线真人客服、华硕产品的维修服务等) 的客务服务提供商。该等服务提供商皆将依循我们的指示于本政策所提及之目的范围内使用自您的个人信息，我们亦会确保其等确实遵守本政策。

3.4 基于法律或保护之目的

如有以下任何一种为遵循法律要求或为保全目的的情形时，我们将揭露、分享您的个人信息的必要项目予第三方:

  • 于有遵守法令、主管机关或司法单位之要求、或为保全法律上的请求、抗辩或为防止诈骗或调查其他不法行为所必要时。
  • 于法律允许之范围内，为保护华硕、我们的服务提供商、使用者或公众之权利、财产及安全不致受危害时。


4. 跨境传输

此段介绍华硕在符合各国个人信息相关法规的情形下，将可能会把您的个人信息传输至您所在国家以外的地区。

您了解并同意当您使用华硕产品或服务或将您的个人信息提供予我们时，您的个人信息皆将可能在我们位处各国的关系企业、服务供货商间进行跨境传输、储存及处理利用。上述跨境传输、储存及处理利用行为皆将遵循本政策及各国个人隐私及个人信息安全之相关法令要求。


5. Cookies及类似技术

此段介绍华硕和第三方合作伙伴使用Cookies和其他类似技术的方式，以及说明您可如何管理Cookies的设定。

我们和第三方合作伙伴使用Cookies(此为储存于您使用之产品上用于纪录信息的小型文本档案)及其它类似技术，例如Web Beacons(网络信标)，来提供华硕产品和服务。如果您造访华硕网域的其中一个网站(含子页面和特定国家/地区的版本)，该网站可能设定以下部份或全部的Cookies及类似技术。
绝大部分的Cookies纪录都仅储存在您的产品上，而不会回传至华硕。只有在少部分的情况下，Cookies纪录可能会回传至华硕，例如当您在ASUS Store购买华硕产品时，华硕为了验证登入ASUS Store的用户和下订单者为同一人，以防止网站诈骗行为的发生，我们可能会透过Cookies将您在登入ASUS Store和下订单时的IP地址分享给华硕。


5.1 我们使用Cookies的方式：

1. 为了满足您使用华硕产品及服务时的线上浏览体验，我们使用以下Cookies，以提供您使用华硕产品及服务所必备的功能：

使用的功能 举例说明
登入与验证 本类型的Cookies可帮助您记住您的登入与验证信息，方便您造访华硕产品及服务，例如ASUS的aticket cookie等。
储存您的偏好设定 本类型的Cookies可将您的偏好设定储存于您的产品中，例如您习惯使用的语言类别、国别或字体;在您再次造访华硕产品及服务时，您即无需再次选择国别、语言类别或字体，例如ASUS的current_site cookie、EntryPage cookie等。
储存您在使用华硕产品及服务所提供的信息 本类型的Cookies可将您在华硕产品及服务中所填写的信息暂时储存在您的产品中，例如您在ASUS Store购买产品时所填写的购买信息，以便记取您欲购买的产品、数量以及您所填写的信息，例如ASUS的count cookie。
安全性 本类型的Cookies可防止网站诈骗行为，保障您在使用华硕产品及服务时的交易安全，例如ASUS的ip_address cookie。为了以上的目的，我们可能会透过此类Cookies将您在登入ASUS Store和您在ASUS Store下订单时的IP地址分享给华硕，以让华硕能够验证登入ASUS Store的用户和在ASUS Store下订单者为同一人。
维持网站流量的稳定性 本类型的Cookies可稳定华硕的网站浏览流量，让您能顺畅地浏览华硕的网站，例如ASUS的BIGipServerNew cookie。

2. 为了统计与使用分析的用途、提供您客制化的广告服务及其他功能的体验，我们使用以下的第三方Cookies，以优化您使用华硕产品及服务时的体验：

使用的功能 举例说明
统计与使用分析 本类型的Cookies可纪录您造访或使用华硕产品及服务的网页及功能的类型、次数与停留时间等信息，例如MIGO Corp.的MIGO cookie、Google Inc.的Google Tag Manager 以及Google Analytics cookie。为了以上目的，我们可能会透过MIGO cookie，将可识别您身份的信息分享给华硕，例如IP地址、您的ASUS会员编号。我们可使用前述信息汇整成统计信息以了解华硕产品及服务的使用效能状况，进而有效地调整或开发新服务或功能。
客制化广告提供 本类型的Cookies可纪录您的使用行为及兴趣点，例如您于华硕产品及服务所浏览过的广告类型;利用该纪录我们及我们的合作伙伴即可提供您感兴趣的营销广告内容。
此外，本类型Cookies也会被用于限制您看到广告的次数以及衡量广告活动的成效性。我们藉由使用广告活动服务，例如Hubrus公司的Hubrus DSP cookie、Google Inc.的Google AdWords cookie和 Google DoubleClick cookie来放置本类型Cookies于华硕产品及服务中。本类型Cookies所搜集得之信息仅供我们及我们的广告活动服务供货商使用。
观看本服务网站的嵌入式YouTube影片 本类型的Cookies由Google Inc.所提供，可让华硕的网站嵌入YouTube影片，以让您透过该网站，便可直接点选收看YouTube影片。



5.2 如何管理Cookies设定

  1. 请注意，您可透过浏览器，依自己的偏好管理您对于Cookies的设定。您可自由地将全部或部分（如第三方的Cookies）的Cookies设定为拒绝使用、删除您浏览器内的Cookies纪录或进行其他相关设定。
  2. 在某些国家，当您第一次浏览华硕的网站时，我们会将简短的Cookies说明文字放在这些网站的最上方横幅中。您可在该横幅中，自由选择开启或拒绝使用上述第三方的Cookies。

5.3 Web Beacons (网络信标)

Web Beacons(网络信标)是指一单一像素(1X1)或GIF、PNG文件的电子图像文件，并可被嵌入任何网页与邮件元素内以纪录或辨识您于华硕产品及服务的使用状况。我们或我们的广告服务提供商可能会使用以上Web Beacons纪录的信息来辨识您是否曾造访华硕产品及服务的特定网页或点击连结，我们也会将Web Beacons嵌入我们的营销信息如营销电子邮件及电子报中以判断您所点阅或链接的讯息内容。我们将利用这些信息来了解您如何使用华硕产品及服务，以增进我们的服务质量与内容。

7. 华硕产品及服务的安全措施

此段介绍我们如何保护您的个人信息，亦提供您如何自行保护您的个人信息的建议。

我们采用适当的技术和组织化的信息安全保护措施，包括对个人信息之搜集、储存、处理以及实体上的安全措施进行内部检查，以防止任何窃取、窜改、泄露或毁损亦以防杜我们储存您个人信息的系统遭受入侵。我们与位于各国的关系企业亦皆以安全的网络专线进行信息传输，当您提供个人信息予我们时，您的个人信息不论是在线或是离线状态都将受到保护。然而，我们无法保证整体网络环境皆是安全无虞的，为保护您的权益，我们建议您:


7.1 妥善保护您的ASUS会员账号，例如：

  1. 创建ASUS会员账号时，使用混合字母及数字之文字符串以创建密码。

  4. 当您的ASUS会员账号密码被盗用时，请立即连络我们，以立即暂停或终止您的ASUS会员账号密码（或其任何部分）权限，并将与您的ASUS会员相关的信息移除。

7.2 您亦应随时更新您的防病毒软件及使用病毒或间谍扫描程序以确保您个人信息的安全。

7.3 若您知悉或发现有任何足以损害华硕产品及服务的技术弱点存在，请立即利用以下电子信箱连络我们:privacy@asus.com
